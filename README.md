# 📘 Εγχειρίδιο Χρήστη Flowdraw

## 1. Εισαγωγή 🎉

Καλώς ήρθατε στο **Flowdraw** - το πιο έξυπνο (και πανέξυπνο) εργαλείο δημιουργίας διαγραμμάτων ροής που θα συναντήσετε! 

Αν ποτέ σκέφτηκετε:
* "Θέλω να φτιάξω ένα διάγραμμα ροής αλλά το Visio μου δίνει πονοκέφαλο"
* "Τα διαγράμματα στο PowerPoint μοιάζουν σαν να τα έφτιαξε ένας χιμπαντζής με κιμωλία"
* "Θέλω κάτι απλό, γρήγορο και που να μην χρειάζεται PhD στην πληροφορική"

...τότε βρήκατε τη λύση! Το Flowdraw είναι το εργαλείο που φανταζόσασταν αλλά ποτέ δεν βρήκατε - απλό, γρήγορο, και με αρκετό χιούμορ για να κάνει τη δουλειά διασκεδαστική.

## 2. Εγκατάσταση 🚀

### 💻 Προϋποθέσεις συστήματος
- **Λειτουργικό σύστημα**: Windows 10 ή νεότερο
- **Αποθήκευση**: Τουλάχιστον 100 MB ελεύθερος χώρος
- **Δικαιώματα**: Συνιστάται εκτέλεση ως διαχειριστής (Administrator) για πλήρη πρόσβαση

### 📦 Οδηγίες εγκατάστασης
1. **Εκτέλεση εγκαταστάτη**: Βρείτε το αρχείο `Flowdraw Setup.exe` και κάντε διπλό κλικ. Αν το Windows σας ρωτήσει "Είστε σίγουρος;", πείτε του "Ναι, είμαι μεγάλος/η τώρα!"

2. **Ακολουθήστε τον οδηγό**: 
   - Πατήστε "Next" όπως ένας επαγγελματίας
   - Διαβάστε την άδεια χρήσης (just kidding, κανείς δεν το κάνει)
   - Επιλέξτε φάκελο εγκατάστασης

3. **Προτεινόμενος φάκελος**: 
   ```
   C:\Flowdraw\
   ```
   *Γιατί το C:\; Γιατί εκεί τα προγράμματα νιώθουν σαν στο σπίτι τους και δεν κάνουν εφιάλτες με δικαιώματα πρόσβασης.*

4. **Συντόμευση**: Σημειώστε το κουτάκι "Create desktop shortcut" για να έχετε το Flowdraw πάντα ένα κλικ μακριά.

5. **Ολοκλήρωση**: Πατήστε "Finish" και περιμένετε τον εορτασμό (ψυχικά - δεν υπάρχουν πραγματικά πυροτεχνήματα).

### ⚙️ Πρώτη εκκίνηση
- **Εκκίνηση**: Βρείτε το Flowdraw στο Start Menu ή στην επιφάνεια εργασίας
- **UAC Alert**: Αν το Windows σας κάνει την ερώτηση "Θέλετε να επιτρέψετε σε αυτήν την εφαρμογή να κάνει αλλαγές;" απαντήστε "Ναι" - δεν πρόκειται να πειράξει τις φωτογραφίες των διακοπών σας
- **Πρώτη εμφάνιση**: Το Flowdraw θα δημιουργήσει τα απαραίτητα αρχεία ρυθμίσεων και θα σας καλωσορίσει με ένα άδειο καμβά, έτοιμο για δημιουργία!

## 3. Ξεκινώντας 🎨

### Η Κύρια Οθόνη
Όταν ανοίγετε το Flowdraw, βλέπετε μια φιλική και οργανωμένη διεπαφή:

**Αριστερά**: Ο καμβάς σας - ο χώρος δημιουργίας, με πλέγμα για ακριβή τοποθέτηση

**Δεξιά**: Ο πλήρης οπλοστάσιο σας:
- 📐 **Παλέτα Σχημάτων**: Process, Decision, Start/End, Data, Document
- 🛠️ **Εργαλεία**: Connect, Edit, Disconnect, Delete, Clear All
- 💾 **Λειτουργίες Αρχείου**: Save, Load, Export PNG
- 🔄 **Ιστορικό**: Undo/Redo για όταν κάνετε "ουπς"
- 📊 **Πληροφορίες**: Τι έχετε επιλέξει και στατιστικά

### Βασικές Ρυθμίσεις
Το Flowdraw έρχεται έτοιμο για δράση! Δεν χρειάζεται πολλές ρυθμίσεις, αλλά μπορείτε να:
- Αλλάξετε μέγεθος παραθύρου (το πρόγραμμα θυμάται την προτίμησή σας)
- Χρησιμοποιήστε τα πλήκτρα συντόμευσης για ταχύτερη εργασία

## 4. Χρήση του Προγράμματος 🎯

### 📝 Βασικές Λειτουργίες

**Προσθήκη Σχημάτων**:
1. Επιλέξτε τύπο σχήματος από την παλέτα
2. Κάντε κλικ στον καμβά για να το τοποθετήσετε
3. Σύρετε για να μετακινήσετε (τα σχήματα δεν δαγκώνουν)

**Σύνδεση Σχημάτων**:
1. Πατήστε το κουμπί "Connect" 
2. Κάντε δεξί κλικ στο πρώτο σχήμα
3. Κάντε δεξί κλικ στο δεύτερο σχήμα
4. Magic! Έχουν συνδεθεί (χωρίς να χρειάζεται κολλητική ταινία)

**Επεξεργασία Κειμένου**:
1. Επιλέξτε σχήμα
2. Πατήστε "Edit" 
3. Γράψτε το κείμενο σας
4. Πατήστε OK και voila!

### ⌨️ Πλήκτρα Συντόμευσης (για τους επαγγελματίες)

| Πλήκτρο | Λειτουργία | Σχόλιο |
|---------|------------|---------|
| `Ctrl + 1` | Process Shape | Για τις βασικές διαδικασίες |
| `Ctrl + 2` | Start/End Shape | Αρχή ή τέλος; You decide! |
| `Ctrl + 3` | Decision Shape | Όταν πρέπει να διαλέξετε |
| `Ctrl + C` | Connect Mode | Για συνδέσεις |
| `Ctrl + D` | Disconnect Mode | Για "διαζύγια" σχημάτων |
| `Delete`   | Delete Selected | Για λάθη που θέλουμε να ξεχάσουμε |
| `Ctrl + Z` | Undo | Το αγαπημένο όλων |
| `Ctrl + Y` | Redo | Όταν το Undo ήταν λάθος |
| `Ctrl + S` | Save Project | Save early, save often! |
| `Ctrl + O` | Load Project | Φέρτε πίσω το έργο σας |
| `Ctrl + E` | Export PNG | Για να το δείξετε και στους φίλους σας |
| `Esc` | Cancel Operation | Όταν αλλάζετε γνώμη |

### 💾 Διαχείριση Αρχείων

**Αποθήκευση Project**:
- Αποθηκεύει ως INI αρχείο
- Διατηρεί όλα τα σχήματα, συνδέσεις και κείμενα
- Μπορείτε να συνεχίστε αργότερα από ακριβώς εκεί που σταματήσατε

**Εξαγωγή PNG**:
- Δημιουργεί εικόνα του διαγράμματος σας
- Τέλειο για reports, παρουσιάσεις, ή απλά για να νιώσετε περήφανοι

## 5. Συχνές Ερωτήσεις (FAQ) ❓

**Ε: Γιατί μου λέει "Access Denied" όταν προσπαθώ να αποθηκεύσω;**
Α: Το Windows μπορεί μερικές φορές να είναι λίγο overprotective. Δοκιμάστε:
- Να εκτελέσετε το πρόγραμμα ως Administrator (Right-click → Run as Administrator)
- Να αποθηκεύσετε σε άλλο φάκελο (π.χ. Documents αντί για C:\)

**Ε: Μπορώ να το εγκαταστήσω σε άλλο drive;**
Α: Φυσικά! Αλλά το προτείνουμε στο C:\ γιατί εκεί έχει λιγότερα "κολλήματα" με δικαιώματα. Αν το βάλετε στο D:\, μπορεί να χρειαστεί να του δώσετε και ένα χάρτη για να βρει το δρόμο του.

**Ε: Γιατί χάνω τις συνδέσεις όταν μετακινώ σχήματα;**
Α: Δεν χάνετε! Οι συνδέσεις παραμένουν και μετακινούνται μαζί με τα σχήματα. Το Flowdraw είναι πιο πιστό και από golden retriever.

**Ε: Πόσα σχήματα μπορώ να βάλω;**
Α: Μέχρι να μην βλέπετε πλέον τον καμβά από πάνω! Αλλά σοβαρά, δεν υπάρχει αυστηρό όριο - απλά η απόδοση μπορεί να επηρεαστεί αν φτάσετε τα χίλια.

## 6. Αντιμετώπιση Προβλημάτων 🛠️

### Κοινά Προβλήματα και Λύσεις

**Το πρόγραμμα δεν ανοίγει:**
- Έλεγχος antivirus (μερικά antivirus κάνουν ζημιά σε AHK προγράμματα)
- Δοκιμάστε να εκτελέσετε ως Administrator
- Ελέγξτε αν το Windows SmartScreen το έχει μπλοκάρει

**Το πρόγραμμα "κολλάει":**
- `Ctrl + Alt + Del` → Task Manager → Βρείτε το Flowdraw → End Task
- Ξανά εκκινήστε το - μερικές φορές όλοι χρειαζόμαστε ένα fresh start

**Χάθηκαν τα projects μου:**
- Τα projects αποθηκεύονται όπου εσείς επιλέξετε
- Προτείνουμε να δημιουργήσετε ένα φάκελο "My Flowdraw Projects" για εύκολη εύρεση

**Καθαρή Επανεγκατάσταση:**
1. Απεγκατάσταση από Control Panel
2. Διαγραφή του φακέλου `C:\Flowdraw\` (αν υπάρχει)
3. Επανεγκατάσταση από το αρχικό setup
